## How To Use
Running:
```
manimgl main.py
```
Saving the video:
```
manimgl main.py -w
```
Opens video after saving:
```
manimgl main.py -w
```
Viewing last frame:
```
manimgl main.py -s
```
Viewing *number* frame:
```
manimgl main.py -n <number>
```
